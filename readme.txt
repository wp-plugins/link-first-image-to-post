Automatically link the first image in each post to that post's permalink.

== Description ==

This plugin will search for the first image of every post on a category, archive or home page listing, then link that image to that post's permalink. This allows your readers to click through to your posts intuitively and easily, meaniing you gain extra pageviews and better statistics.

A super lightweight plugin which has no options. Simply install, activate and you're good to go!

WordPress plugin developed by [pipdig](http://www.pipdig.co?utm_source=wordpress&utm_medium=wprepo&utm_campaign=linkedimg).

== Installation ==

1. Upload `link-first-image-to-post` folder to the `/wp-content/plugins/` directory.
2. Activate the `Link First Image to Post` plugin through the 'Plugins' menu in WordPress.
3. That's it, you're done! The first image on each post will now link to that post.
